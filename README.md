## **README文件**

本光盘作为自定义工具打包构建专用项目，**我们希望一个包囊括用于创作的所有必要的文件**。

## 概述

本项目主要为打包自己的作品提供了环境，只需把压缩包放到必要的位置就可以令这个项目起作用。

但是，使用该项目打包你自己的作品时，务必遵守所在地区的法律法规。

## 目标

我们不是因为要让萌新学会挂载镜像文件，而是要让萌新知道CD安装包也有他自己的真正意义。

***也许在国家法律范围内，这个项目所成的打包并不能让你玩到诸如《千恋万花》、《若能与你再次相见》之类的GalGame，但是……***

***有时候你看到的带密码的压缩包往往会带有信息差……***

顺带提一嘴，iso文件在Windows 10/11操作系统可以直接挂载，不需要第三方软件。

## 为何不用VHD文件？

有关vhd文件相关信息可以在[这里](https://learn.microsoft.com/zh-cn/windows/win32/vstor/about-vhd)找到。
